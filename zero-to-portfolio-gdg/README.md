# GDGoC Event Repo - Zero to Portfolio

Welcome to the **Zero to Portfolio** repository!

This project is part of a GDGoC (Google Developer Groups on Campus) event where we teach you:
1.  **How to build your first portfolio website** using modern HTML, CSS, and JavaScript.
2.  **How scalable systems work** and best practices for structuring your web projects.

## Project Overview

This portfolio features:
-   **Modern UI/UX**: Clean black and white aesthetic with typing indicators and smooth animations.
-   **Fully Responsive**: Works seamlessly on desktop and mobile devices.
-   **Interactive Elements**: Hover effects, scroll animations, and a dynamic navigation menu.

## Getting Started

1.  Clone the repository.
2.  Open `index.html` in your browser to view the portfolio.
3.  Customize the content in `index.html` and styles in `style.css` to make it your own!

Happy coding!
